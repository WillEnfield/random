.txt file runner documentation

The .txt file runner runs .txt files as code, in a special coding language described in these docs.

Syntax

Each statement must end with a ;. Any numbers or equasions must be encased in a <>

Statements

Statements are ususaly single lines of code, terminated by ;. All statements are described below.

  • var a = b - Sets the variable 'a' to 'b'. If 'a' does not yet exist, creates 'a'.
